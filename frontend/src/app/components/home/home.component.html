Welcome to the Loan Origination System

This is the assessment project for the Frontend Developer position.

Getting Started

  1. Review the TASKS.md file for detailed task requirements
  2. Complete the three assessment tasks:
    • Task 1: Loan List Component
    • Task 2: Loan Application Form
    • Task 3: Loan Details & Performance Optimization
  3. Ensure the backend server is running on http://localhost:3000
  4. Start the Angular development server with ng serve

API Endpoints

The backend provides the following endpoints:

  • GET /api/loans - Get all loans
  • GET /api/loans/:id - Get a specific loan
  • POST /api/loans - Create a new loan
  • PUT /api/loans/:id - Update a loan
  • DELETE /api/loans/:id - Delete a loan
  • GET /api/borrowers - Get all borrowers
  • GET /api/borrowers/:id - Get a specific borrower
